# Arrays - Basics (Java)

## Topics Covered
- Array Declaration
- Traversal
- Find Max/Min
- Reverse Array
- Count Frequency

## Time Complexities
- Access: O(1)
- Traversal: O(n)
- Search: O(n)

## Problems Solved
- Find Largest Element
- Find Second Largest
- Reverse Array
- Check Sorted Array

## Key Learnings
- Arrays use contiguous memory
- Indexing starts from 0
- Two pointer technique is important
