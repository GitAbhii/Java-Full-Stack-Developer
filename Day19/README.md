# Day19 - Exception Handling (Java)

This repository demonstrates exception handling in Java using try-catch blocks.

## Programs Included

### TryCatchExample.java
Shows how to handle arithmetic exceptions using try and catch.

### MultipleCatchExample.java
Demonstrates handling multiple exceptions using multiple catch blocks.

## Concepts Covered
- try block
- catch block
- handling runtime errors
- multiple catch statements

## Goal
To understand how Java prevents program crashes using exception handling.
