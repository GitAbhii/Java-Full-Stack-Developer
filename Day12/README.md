# Day12 - Classes and Objects (Java)

This repository contains beginner-level Java programs demonstrating the concept of Classes and Objects.

## Programs Included
- Student Class Example
- Employee Class Example
- Car Class Example
- Book Class Example
- Mobile Class Example

## Concepts Covered
- Class creation
- Object creation
- Accessing variables using objects
- Calling class methods

## Goal
To build a strong foundation in Object-Oriented Programming (OOP) concepts for backend developer preparation.
