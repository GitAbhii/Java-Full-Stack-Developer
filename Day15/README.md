# Day15 - Encapsulation (Java)

This repository contains Java programs demonstrating the concept of Encapsulation.

## Programs Included
- Student Encapsulation
- Bank Account Encapsulation
- Employee Encapsulation

## Concepts Covered
- private variables
- Getter and Setter methods
- Data hiding

## Goal
To understand how data can be protected using private access modifiers and accessed using public methods.
