# Day16 - Polymorphism (Java)

This repository contains Java programs demonstrating the concept of polymorphism.

## Concepts Covered
- Method Overloading (Compile-time Polymorphism)
- Method Overriding (Runtime Polymorphism)

## Programs
Method Overloading (Compile-time Polymorphism)
Method Overriding (Runtime Polymorphism)
Another Overriding Example

## Goal
To understand how the same method name can perform different tasks based on parameters or class inheritance.
